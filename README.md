# QuickDraw DATASET + Alarm







## 프로젝트 소개

Google quickdraw dataset을 이용한 AI로 이미지 분류 기능과 알람 앱을 결합한 서비스


## 개발기간
2023.7.30 ~ 2023.8.7 (1주)


## 개발환경
react-native 0.67.1

IDE: vscode, android studio

server: nodejs express, AWS EC2
